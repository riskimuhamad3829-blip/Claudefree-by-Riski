📖 About

ClaudeFree is a lightweight command-line client for accessing Claude Sonnet and Claude Opus through a public API.

«⚠️ Warning

This project relies on a public API. The API may occasionally become unavailable, rate-limited, or return temporary errors.»

---

✨ Features

Feature| Description
🤖 Claude Sonnet| Chat with Claude Sonnet
🧠 Claude Opus| Chat with Claude Opus
⚡ Lightweight| Minimal resource usage
🐧 Linux| Full Linux support
📱 Termux| Android Termux compatible

---

📦 Installation

Clone Repository

git clone https://github.com/riskimuhamad3829-blip/Claudefree-by-Riski.git
cd Claudefree-by-Riski

Run

chmod +x claudefree
./claudefree
