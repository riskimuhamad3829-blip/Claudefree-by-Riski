ClaudeFree by Riski

Free CLI for Claude Sonnet & Claude Opus powered by a public API.

«⚠️ Warning

This project uses a public API. The API may occasionally be unavailable, rate-limited, or return errors. If that happens, wait a few minutes and try again.»

Features

- 🤖 Claude Sonnet
- 🧠 Claude Opus
- ⚡ Fast and lightweight
- 🐧 Linux & Termux support

Installation

git clone https://github.com/riskimuhamad3829-blip/Claudefree-by-Riski.git
cd Claudefree-by-Riski
chmod +x claudefree
./claudefree
